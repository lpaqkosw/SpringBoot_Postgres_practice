# A simple ToDo list

### Objective
* For personal use(for now)
* To keep in track of what I will do, what I am doing, and what I have done
* To try out the famous reactjs
* To try out material-ui

### Frameworks, Tools Used
* Reactjs & React-router
* Spring Boot 
* PostgreSQL
